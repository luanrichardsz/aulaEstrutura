## Operador de Atribuição
Como na maioria das linguagens, o sinal de igualdade é utilizado para representar a atribuição.

```c
#include <stdio.h>
int main() {
 int x,y,z;
 x = 5;
 y = z = 10;
 printf("X=%d, Y=%d, Z=%d \n",x,y,z);

 saida(X=5, Y=10, Z=10)
 return 0;
}
```
